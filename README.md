# project-1d-cli-files

CLI for create files

## Install

* GLOBAL
```
npm project-1d-files-cli -g
```

* LOCAL
```
npm project-1d-files-cli
```

## Use

* GLOBAL
```
// Launch in terminal
project-1d-files-cli
```

* LOCAL
```

// Add to package.json
"scripts": {
  "project-1d-cli-files": "./node_modules/.bin/project-1d-cli-files"
}

// Launch in terminal
npm run project-1d-files-cli

```
